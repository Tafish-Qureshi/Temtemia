# Temtemia

## Introductie
Dit is een project waarmee ik bezig ben van een spel waar ik eigenlijk een soort verbeterde pokedex voor maak voor spel die ik zelf vrij leuk vindt om te spelen

## repo clonen 
als je deze project via https://github.com/Tafish-Qureshi/Temtemia wilt clonen dan is het volgende wat je kunt doen 

Stap 1:  De blauwe knop genaamd code
Als je bij de repository erbij bent moet op de blauwe knop met code erop klikken dan krijg je het volgende te zien:
```
https://github.com/Tafish-Qureshi/Temtemia.git
```

Dan moet je deze kopieren en dan naar stap 2 gaan

Stap 2: plek waar je de repo wilt opslaan

Als je de repository link hebt gekopieerd dan ga je naar je bureaublad,
dan als je op je bureablad bent en je hebt een mapje genaamd repository (mag ook gewoon direct op je bureaublad je hoeft niet perse een mapje te hebben) dan doe je linkermuisklik op je bureablad of op de plek waar je bent
je powershell of je command prompt open en dan typ je het volgende 

```
git clone https://github.com/Tafish-Qureshi/Temtemia.git
```

dan heb je eigenlijk de map in principe gecloned

## Live server

Als je de repo hebt gecloned dan kan je ook met cd (Change directory) naar de map gaan waar je het hebt gecloned en kan je in principe ook het volgende doen of als het op je bureablad is kan je het ook direct openen
en naar de index.html gaan om de website te openen. Maar als je het via VS code studio wilt doen doe je het volgende:
```
cd naar de map waar je het hebt opgeslagen
```

dan als je in je map bent:
```
code .
```

daarmee open je eigenlijk je code editor en als je de live server extensie hebt dan kun je in principe op die manier de index.html openen
